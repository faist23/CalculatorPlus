# Casio fx-991EX CLASSWIZ — Complete Key Reference

A comprehensive specification of every key on the Casio fx-991EX CLASSWIZ scientific calculator, including primary labels, all secondary labels, and the exact color of each label.

---

## Color Legend

| Color | Meaning | Access |
|---|---|---|
| **White** | Primary function | Direct keypress |
| **Yellow** | Shifted function | Press `SHIFT` first |
| **Pink / Magenta** | Alpha function (variables, constants, separators) | Press `ALPHA` first |
| **Blue** | Base-N mode function | Active only in BASE-N mode |
| **Blue key body** | Editing / clear keys (DEL, AC) | Direct keypress |

## Special Color Notes

1. **Yellow vs. Pink distinction** — Yellow is always math/computational; pink is always *variable/text-entry*. The pink `=` and `:` symbols above the `CALC`, `INTEGRAL`, and `VAR_X` keys are for typing those characters into equations and spreadsheet ranges, not for performing operations.

2. **Italic pink letters** (*i*, *x*, *y*, *e*) — Italics indicate mathematical constants/variables (imaginary unit, algebraic variables, Euler's number) as distinct from the upright pink capitals (A–F, M) which are storage variables.

3. **The "x" appears twice** — Once as a white italic primary key (Row 2, the variable for equations/integrals/summations), and once as a pink italic ALPHA label above the `)` key (Row 5, the algebraic variable for spreadsheet/table use).

4. **Blue BASE-N labels** (DEC, HEX, BIN, OCT) appear only on the `x²`, `x■`, `log■`, and `ln` keys — these activate only when BASE-N mode is selected from the MENU.

5. **Casio branding** — "fx-991EX" is in white, "CLASSWIZ" is in pink/magenta on the body above the screen.

6. **SETUP label position** — The yellow SETUP label appears *next to* the MENU key on the calculator body (not above as a typical shifted label), because the cursor pad sits directly above MENU and there is no room for an above-key label.
